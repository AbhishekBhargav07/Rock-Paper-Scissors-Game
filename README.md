# ✊✋✌️ Rock Paper Scissors Game

A responsive and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Challenge the computer in the classic game with real-time score tracking, smooth gameplay, and an intuitive user interface.

## 🚀 Features

* 🎮 Play against the computer
* 🧠 Random computer move generation
* 📊 Live score tracking
* 🏆 Instant win, lose, or draw results
* 📱 Responsive design for desktop and mobile devices
* 🎨 Clean and modern user interface
* ⚡ Fast and lightweight with no external libraries

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)

## 📚 What I Learned

* DOM Manipulation
* Event Handling
* Random Number Generation
* Conditional Logic
* Functions and Code Organization
* Responsive Web Design

This project helped strengthen my JavaScript fundamentals by implementing game logic, user interaction, and dynamic UI updates in a real-world application.
